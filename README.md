
## SIP仿真工具 可以直接将完整的SIP消息文本发送到指定端口

# 优点：
之前的SIP测试工具只能通过XML文件定制SIP消息，对于不熟悉SIP协议的人很难上手。本工具可以直接将已有的SIP消息通过SOCKET发送出去，主要用于SIP协议对接调试阶段。让码农只关注产品BUG，而不再浪费太多的精力在问题复现、环境整理上面。

# 当前支持的功能：
通过UDP发送SIP报文到指定IP；完成端到端呼叫（UAC\UAS)

# 使用范围：
使用SIP协议完成呼叫。
